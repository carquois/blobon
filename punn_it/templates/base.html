{% trans "Bienvenue sur le nouveau Check Donc Ça." %}

Check Donc Ça est basé sur un principe simple, mais puissant : le partage.

Partage de contenu = partage de revenu

Imaginez : faire de l'argent dans le confort de votre maison avec votre contenu Web. Check Donc Ça offre une manière facile et gratuite de générer des revenus grâce à votre contenu en ligne.

  1. {% trans "Créez votre compte" %}
  2. {% trans "Mettez votre profil à jour" %}
  3. {% trans "Créez du contenu" %}
  4. {% trans "Consultez vos revenus" %}
À quelle fréquence payons-nous?

Nous payons quand votre compte atteint 100$. Nous nous sommes basés sur Google AdSense pour fixer la limite. Comment payons-nous?

Encore une fois, comme AdSense, nous offrons deux choix de paiement :

  1. Par chèque qui vous sera envoyé par la poste
  2. Par virement électronique qui sera déposé directement dans votre compte

Vous pourrez mettre à jour vos informations de paiement lorsque vous aurez atteint le montant du premier virement (100$).

Quand payons-nous?

Nous payons une fois par mois, lorsque le compte dépasse 100$.

Comment optimiser vos revenus?
  • Publiez de manière constante. Ne vous gênez pas sur la fréquence.
  • Ajoutez une catégorie. Les catégories fonctionnent un peu comme des sous-communautés (subreddit). Quoiqu'il ne soit pas obligatoire d'assigner une catégorie, il est toutefois fortement recommandé de le faire. De cette manière, vous serez «repris» sur ces pages, ce qui augmentera le trafic sur votre contenu.
  • Partagez sur vos réseaux sociaux. Les réseaux sociaux sont la source principale de visites sur le site. Vous pouvez aussi inviter vos amis à participer.
  • Soyez actif dans la communauté. Votez, commentez, participez. L'interaction est à la base de l'intérêt. Nous en prenons compte dans notre calcul de partage des revenus.
  • Citez votre source. Les créateurs de contenu méritent le trafic qu'ils obtiendront si vous partagez leur contenu ici. Soyez gentil et citez-les.

Et n'essayez pas de spammer. J'vous l'dis là, j'suis pas épais. J'vais vous trouver et vous bannir. ;-)
